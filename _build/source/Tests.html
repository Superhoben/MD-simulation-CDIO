Tests.unittests module¶

class Tests.unittests.UnitTests(methodName='runTest')[source]¶

Bases: TestCase

Definitions of unittests.

Different tests can be added here from different files. To add a new test, import the functions from the file that is to be tested and then write a new test function below the existing ones.

See example tests below.
